# Praticas-DevOps-2026

Repositório desenvolvido para a disciplina de Laboratório de Programação para Operação de Infraestrutura de Redes de Computadores, ministrada pelo Professor Eder Pereira dos Santos.

## Objetivo

Este repositório reúne práticas desenvolvidas ao longo da disciplina, com foco em comandos Linux, scripts Bash, automação de tarefas, manipulação de arquivos, análise de logs e organização de projetos voltados para DevOps.

As atividades foram desenvolvidas e testadas em ambiente Linux.

## Estrutura do Repositório

- `server/`: arquivos relacionados à configuração do servidor e crontab.
- `backup/`: espaço reservado para scripts de backup.
- `utils/`: scripts utilitários e listas de exercícios.
- `monitoring/`: scripts de monitoramento do sistema.
- `ci_cd/`: espaço reservado para práticas de integração contínua.
- `logs/`: arquivos de log e saídas geradas pelos scripts.
- `provisioning/`: scripts relacionados ao provisionamento de infraestrutura.

## Atividades em `utils/`

- `devops_ex1/`: scripts introdutórios em Shell Script.
- `devops_ex2/`: scripts com condicionais, operadores e argumentos.
- `devops_ex3/`: scripts para manipulação de arquivos, logs e informações do sistema.
- `devops_ex4/`: scripts com funções, expressões regulares e validação de dados.
- `essential_commands.sh`: script interativo com comandos essenciais para DevOps.
- `mostrar_tudo.sh`: script auxiliar para exibir o conteúdo dos exercícios.
